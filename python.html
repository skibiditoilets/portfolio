CodeCompanion

I built a python based app using Streamlit that helps people learn how to code with the help of generative AI.

Here is a brief introduction of our application.

Origin

Our school organised a machine learning workshop in collaboration with Republic Polytechnic. Over the 5 day course, we learnt how to use generative AI to create web applications.

CodeCompanion was an idea that we had brainstormed during the workshop, but never came to fruition there.

Features

Learning

It allows the user to first learn about Python. If the user needs more in depth explaination, we use generative AI features to create explainations for all the code.

Questions

After the user has gotten a basic grasp of the python concepts, they can test their skills on the question generator where users can generate questions based on a particular topic and difficulty.

Marking

Finally, users are able to submit their code and recieve feedback based on their code.

Unfortunately, due to the lack of funding, we are unable to provide ChatGPT API keys (which is our generative AI machine) which are needed to allow the app to run. Everytime a response a generated, a small fee is deducted, therefore disallowing us to deploy the app to the public. However, a copy of the app's code can be downloaded here.
